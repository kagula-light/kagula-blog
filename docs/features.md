# 功能范围

## 角色能力矩阵

| 能力 | 访客 | USER | ADMIN |
| --- | --- | --- | --- |
| 阅读公开文章与热点 | 是 | 是 | 是 |
| 搜索、分类、标签和归档 | 是 | 是 | 是 |
| 注册和登录 | 是 | 已登录 | 已登录 |
| 点赞与收藏 | 否 | 是 | 是 |
| 提交评论 | 否 | 是，需审核 | 是，仍记录审核状态 |
| 查看个人收藏和评论 | 否 | 是 | 是 |
| 文章与媒体管理 | 否 | 否 | 是 |
| 评论审核 | 否 | 否 | 是 |
| 用户管理 | 否 | 否 | 是 |
| 热点审核与来源管理 | 否 | 否 | 是 |
| 站点配置与审计查看 | 否 | 否 | 是 |

禁言和封禁状态会进一步限制 USER 能力。

## 首页

- 每个浏览会话首次访问展示全屏欢迎场景。
- 显示北京时间日期、时间、站点名、作者和短句。
- 提供跳过入口，键盘可操作。
- 欢迎场景结束后进入主站，不再次打断同一会话。
- 主站包含导航、搜索、热门关键词、推荐轮播和文章流。
- 文章流支持最新发布、最近更新、最多浏览、最多点赞和最多评论。
- 首页提供今日热榜入口和有限预览。
- 未登录用户可以进入注册与登录。

## Live2D 看板娘

- 看板娘使用原创“神乐静无月”Live2D 模型，不使用授权不明的动漫或游戏模型。
- 静态海报先于运行时显示，Live2D 加载失败不影响导航、阅读和互动。
- 桌面端在欢迎场景结束且浏览器空闲后按需加载；移动端默认只显示启动入口，由用户点击加载。
- 用户可以让看板娘休眠或关闭；关闭偏好保存在本地，之后不自动打扰。
- `prefers-reduced-motion` 下不自动加载模型，只保留静态海报或显式启动入口。
- 看板娘可提供有限的问候、页面状态提示和切换动作，不提供聊天、语音识别或远程脚本能力。
- 固定位置不得覆盖正文、目录、评论输入、Cookie 提示或移动端主要操作。

完整行为、技术和授权边界见 [Live2D 看板娘](live2d-mascot.md)。

## 文章发现

- 全文搜索标题、摘要、正文纯文本、分类和标签。
- 分类、标签、年度/月度归档页面。
- 热门文章、相关文章和上一篇/下一篇。
- RSS、站点地图、Canonical URL、Open Graph 和结构化数据。
- 搜索结果不泄露草稿、定时文章或未审核评论。

## 文章阅读

- 顶部封面和面包屑。
- 作者、发布时间、更新时间、字数、阅读时长和分类。
- 手动填写的 AI 摘要卡片。
- Markdown 标题、列表、表格、引用、代码块、图片和链接。
- 桌面端右侧吸附作者卡与目录。
- 移动端正文单列，目录通过按钮打开。
- 文末标签、许可说明、相关文章和评论区。
- 登录用户可以点赞、收藏和提交评论。

## 认证与用户中心

当前已实现：

- 用户名密码登录、统一凭据错误、Redis 失败限流和安全 `next` 路径校验。
- 数据库摘要会话、HttpOnly/SameSite Cookie、单会话退出和 BANNED 即时失效。
- ADMIN 登录进入 `/admin`，USER 登录返回公开首页；未登录或 USER 不能进入后台。
- 一次性管理员 bootstrap 与同名管理员凭据轮换。

尚未实现：公开注册、用户资料、头像、收藏/评论列表、管理员重置密码和 Turnstile。

- 用户名密码注册。
- 登录、退出和会话过期。
- 注册与登录限流。
- Cloudflare Turnstile 防止自动化注册。
- 用户资料、头像、收藏列表和评论记录。
- V1 不提供自助找回密码；管理员可以重置密码。
- 邮箱和外部账号数据结构只为未来扩展预留。

## 评论

- 评论提交后明确提示“等待审核”。
- 管理员可通过、拒绝和软删除。
- 审核操作记录操作者和时间。
- 被禁言用户不能提交新评论。
- 被封禁用户的现有会话失效。
- V1 不做嵌套回复、匿名评论和实时通知。

## 管理后台

后台与公开站点共享品牌标识，但使用高信息密度、低装饰的工作界面。

当前已实现受保护后台壳、当前管理员显示、退出、文章工作台和媒体入口。仪表盘统计、评论、用户、热点、设置与完整审计查看仍为后续范围。

### 仪表盘

- 已发布、草稿和定时文章数量。
- 待审核评论和待审核热点数量。
- 用户状态分布。
- 热点来源最近成功与失败。
- 最近管理操作。

### 文章

- 已实现创建、编辑、预览和归档；复制仍为后续范围。
- 已实现 Markdown 编辑器、服务端清理预览和显式不可变修订。
- 已实现分类、标签、封面元数据、手动摘要和 SEO 字段。
- 已实现立即发布、定时发布和 Worker 到期发布。

### 媒体

- 已实现图片上传、元数据列表和 R2 对象边界。
- 已实现 JPEG/PNG/WebP/AVIF 文件头、MIME、大小和尺寸校验；SVG 默认拒绝。
- 响应式变体、引用位置查看和删除操作仍为后续范围。

### 评论

- 待审核、已通过、已拒绝和已删除视图。
- 单条与批量审核。
- 从评论进入用户详情。

### 用户

- 搜索用户、查看状态和互动记录。
- 禁言、封禁、解封和强制退出。
- 管理员重置密码。
- V1 不允许从后台将普通用户提升为多种细粒度角色。

### 热点

- 按来源、状态和时间筛选候选。
- 审核、拒绝、手动补录和调整公开顺序。
- 查看来源最后成功、失败和耗时。
- 单来源重试和暂停。
- 查看每日归档。

### 设置与审计

- 品牌、介绍、导航和默认封面。
- 功能开关只覆盖明确允许的选项。
- 查看管理员操作日志。

## 每日热点

- 来源：GitHub Trending、Hacker News、Bilibili、微博热搜、百度热搜。
- 默认每 30 分钟采集。
- 每个来源独立超时和失败。
- 只保存标题、排名、来源、原文链接和采集时间。
- 候选必须经管理员批准。
- 前台展示当前热榜和按日期归档。
- 归档日期按北京时间。
- 公开链接明确标识来源并跳转原站。

## 站点暂不包含

- 邮箱验证、邮件通知和找回密码。
- 社交登录。
- 用户投稿和多作者发布。
- 签到、积分、等级、关注和排行榜。
- 自动 AI 摘要。
- 付费、赞赏和订阅。
- 私信、实时聊天和评论通知。
